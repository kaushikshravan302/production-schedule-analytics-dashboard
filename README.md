# Production Schedule Analytics Dashboard

## Project Overview
This project focuses on analyzing manufacturing operations data using Power BI. The dashboard tracks production schedules, delays, manpower allocation, department-wise production, and operational insights.

---

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

---

## Key Features
- KPI monitoring
- Production trend analysis
- Delay analysis
- Semantic data modeling
- Data cleaning and transformation
- Unknown/inconsistent record handling
- Interactive filtering

---

## Data Cleaning Tasks Performed
- Removed duplicates
- Handled null values
- Identified unknown categories
- Managed inconsistent dimension relationships
- Validated operational records

## Key Insights

- Adhesive A contributes nearly 50% of production quantity.
